# Customer Churn Cloud Project

## Project Overview
This project analyzes the Telco Customer Churn dataset to identify churn patterns, perform customer segmentation, and build a predictive model for churn classification.

## Objectives
- Perform data cleaning and preprocessing
- Conduct exploratory data analysis
- Handle missing values and outliers
- Apply clustering for customer segmentation
- Build a churn prediction model
- Evaluate model performance
- Integrate Azure cloud services
- Track project progress using GitHub

## Dataset
- Dataset: Telco Customer Churn
- Source: Kaggle
- File: `data/Telco-Customer-Churn.csv`

## Project Structure
- `data/` - dataset files
- `notebooks/` - Jupyter notebooks
- `outputs/` - charts and results
- `screenshots/` - Azure and GitHub screenshots

## Tools and Technologies
- Python
- Pandas
- Matplotlib
- Scikit-learn
- Azure Blob Storage
- Azure Machine Learning
- GitHub

## Cloud Setup
- Dataset uploaded to Azure Blob Storage
- Analysis environment managed through Azure Machine Learning
- Cloud computation performed using Azure ML compute instance

## Models Used
- K-Means Clustering
- Random Forest Classifier

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- Confusion Matrix
